# MENA Pop and Maqam-Adjacent Fusion

> **Snapshot note:** Last content review snapshot: 2026-04-26. Use this file for stable compositional conventions, but verify current artists, charts, platform norms, and regional scenes with active web research. See `../research/web-music-trend-research.md` and `../research/reference-track-digging.md`.

This file covers composition-facing guidance for Middle Eastern and North African pop/fusion requests: Arabic pop, Turkish pop, Persian pop, Levantine/Gulf/North African influences, maqam-adjacent color, and Western pop/EDM/hip-hop crossovers. It is not a full guide to Arabic, Turkish, Persian, Kurdish, Amazigh, or other regional classical/folk traditions.

## When to consult this file

- The user wants Arabic pop, Turkish pop, Persian pop, MENA club/pop, or maqam-adjacent color
- The user says "Middle Eastern feel" and needs a safer, more specific translation
- The user wants microtonal/modal melody in a pop context
- The user asks for current regional pop references; pair with active web research

## Core warnings

- **Maqam is not just a scale.** It includes characteristic phrases, melodic direction, modulation behavior, ornament, tuning tendencies, and aesthetic conventions.
- **Quarter-tones are not a shortcut.** Some maqamat use intervals that differ from 12-TET, but equal 24-TET is only an approximation in many contexts.
- **MENA is not one sound.** Egyptian, Lebanese, Gulf, Turkish, Persian, Maghrebi, Kurdish, Amazigh, and diaspora pop scenes differ.
- Avoid vague cultural shorthand. Name the actual musical variable: augmented second, drone, ornament, iqa'/usul-inspired rhythm, oud-like pluck, maqam Hijaz-adjacent tetrachord, etc.

## Safer translation of vague requests

| User says | Translate to |
|---|---|
| "Middle Eastern" | Which region or reference? Arabic pop, Turkish pop, Persian ballad, Levantine dance, Gulf rhythm, Maghrebi chaabi/raï-adjacent, etc. |
| "Arabic scale" | Ask for maqam family or use "Hijaz-adjacent / Bayati-adjacent / Rast-adjacent color" cautiously |
| "Desert / mysterious" | Drone, sparse percussion, low register, ornamented melody — avoid stereotypes |
| "Belly dance feel" | Often a dance/rhythm/arrangement request; identify rhythm and instrumentation |

## Melodic language

For pop-fusion, use maqam-adjacent constraints without overclaiming.

Useful abstractions:

| Idea | Pop adaptation |
|---|---|
| Jins / tetrachord color | Build hook around a 4–5 note cell with a clear characteristic interval |
| Sayr / melodic path | Decide whether melody opens low, rises gradually, or circles a focal pitch |
| Ornament | Slides, turns, mordents, grace notes, bends, melisma |
| Drone / tonic emphasis | Keep a stable center while melody supplies color |
| Modulation | Move to a related color in bridge or instrumental break, then return |

### Common 12-TET approximations

Use these as **approximations**, not authentic maqam definitions.

| Color | 12-TET pitch set in C | Caution |
|---|---|---|
| Hijaz-adjacent | C-D♭-E-F-G-A♭-B♭ | Strong augmented-second color; often overused as stereotype |
| Phrygian / ♭2 color | C-D♭-E♭-F-G-A♭-B♭ | Darker, less augmented-second bite |
| Harmonic minor | C-D-E♭-F-G-A♭-B | Western minor dominant color; not automatically Arabic/Turkish/Persian |
| Rast-adjacent approximation | C-D-E-ish-F-G-A-B♭-ish | Needs non-12-TET/tuning nuance in traditional contexts |
| Bayati-adjacent approximation | C-D-half-flat-ish-E♭-F-G-A♭-B♭ | 12-TET cannot capture the characteristic second degree well |

If microtonal nuance matters, consult `../techniques/microtonal.md` and current/practitioner sources.

## Harmony

Traditional maqam-based music is melody-centered rather than Western chord-progression-centered, but pop/fusion often uses chords.

Safer harmonic approaches:

| Goal | Approach |
|---|---|
| Preserve modal melody | Use drone/pedal + sparse chords |
| Pop accessibility | Use i-♭VII-♭VI-♭VII, i-iv-V, or I-♭VII-IV loops |
| Hijaz-adjacent drama | Use i-♭II or i-♭II-V-i carefully |
| Ballad color | Use minor add9, sus, maj7, and slow harmonic rhythm |
| Dance/club | Keep harmony simple; let rhythm, bass, and ornament carry identity |

Avoid forcing every melody note into a Western triad. Chords can flatten the modal identity if they move too often.

## Rhythm

MENA rhythms vary widely. Do not use one "Arab beat" for everything.

Composition-facing options:

| Rhythm idea | Use |
|---|---|
| Maqsum-like 4/4 | Versatile Arabic pop/dance frame |
| Saidi-like feel | Earthy, dance-oriented, often associated with upper Egyptian style |
| Malfuf-like 2/4 | Fast entrance, procession, dance energy |
| 6/8 or 12/8 | Gulf, North African, or pop-ballad contexts depending on style |
| Turkish usul-inspired cycles | Longer accent cycles, odd meters, asymmetry in some styles |
| Pop/EDM hybrid | Four-on-floor or trap drums with regional percussion layers |

If the user names a specific dance, region, or tradition, search and verify rather than guessing.

## Arrangement and instrumentation

Common color sources:

- oud-like plucked lines
- qanun-like tremolo/ornament figures
- ney/flute-like breathy lead
- violin/strings with slides and ornaments
- darbuka/doumbek/riq/frame-drum-inspired percussion
- synth lead with pitch bends
- call-response backing vocals
- drone or pedal bass

Do not rely on one sample-pack instrument as a cultural proxy. The melodic phrase, ornament, rhythm, and form matter more than the patch name.

## Vocal writing

Useful moves:

- allow melisma or ornament at phrase endings
- write space for breath and expressive turns
- place important words on sustained notes
- avoid over-harmonizing ornamented lines
- consider language stress, vowel length, and dialect-specific pronunciation
- use call-response or unison group vocals for dance hooks

## Form templates

### Arabic/Turkish/Persian pop ballad-inspired

```text
Instrumental motif → Verse → Pre-lift → Chorus → Instrumental answer → Verse 2 → Chorus → Bridge/modulation → Final chorus
```

### MENA club-pop fusion

```text
Intro percussion/motif → Vocal hook → Beat drop → Verse → Chorus/drop → Instrumental riff → Verse/rap → Final hook
```

### Maqam-adjacent instrumental cue

```text
Drone + free ornament → pulse enters → 4-bar motif → sequence/transposition → breakdown → return to tonic motif
```

## Quick recipes

### Hijaz-adjacent pop hook

```text
Center: C
Melody cell: C-D♭-E-F-E-D♭-C
Harmony: | Cm | D♭ | Cm | B♭ |
Arrangement: drone + plucked ostinato + hand percussion + modern sub
Caution: do not claim full maqam Hijaz authenticity without phrase/tuning study.
```

### Modal dance cue

```text
Rhythm: maqsum-like 4/4 frame
Bass: tonic pedal with octave movement
Melody: short call phrase, then group response
Form: 8-bar build, 8-bar full groove, 4-bar breakdown, return
```

### Ballad color

```text
Loop: | Am(add9) | G | Fmaj7 | E7sus E7 |
Topline: narrow verse, melismatic answer phrase, chorus peak on ^5 or ^♭6
Texture: soft strings + plucked motif + restrained percussion
```

## Common mistakes

- Treating "Phrygian dominant" as a complete MENA music toolkit
- Calling maqam a scale and ignoring melodic path/ornament
- Using quarter-tones randomly
- Over-chording a modal melody
- Using stereotyped "desert" language or threat coding
- Ignoring regional and language differences
- Copying a current pop hook because the surface scale matches
