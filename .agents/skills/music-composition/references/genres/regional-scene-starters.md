# Regional Scene Starter Map

> **Snapshot note:** Last content review snapshot: 2026-04-27. Regional scenes and genre labels change quickly. Use this file as a dispatch map, then verify current references with active web research and local-language sources when possible.

This file collects regional popular-music scenes that are important enough to recognize but not yet covered by a full dedicated guide in this skill. It is not a history. It is a composition-facing map: what to listen for, which craft variables matter, and where the agent should be careful.

Use this file when the user asks about:

- regional Mexican, corridos tumbados, norteño/banda-adjacent pop
- sertanejo, pagode, arrocha, piseiro, Brazilian regional pop beyond the main Brazilian guide
- raï, Maghreb pop, Arabic club-pop subscenes
- Turkish arabesk-pop, Anatolian pop-adjacent modern styles
- Punjabi pop, bhangra-pop, filmi/non-filmi crossovers
- gqom, singeli, gengetone, coupé-décalé, kuduro, or other local dance forms
- any regional scene that is too specific for a general genre file but too important to ignore

For microgenres and internet tags, use `minor-and-hybrid-genres.md`. For broader trend analysis, use `../research/regional-trend-evolution-analysis.md`.

## Core principle

Do not reduce a regional scene to one rhythm, one instrument, or one scale. Start by asking:

```text
Is the user asking for the current scene, a traditional root, a pop crossover, a dance function, or a reference-track lane?
```

Then extract craft variables:

| Variable | What to check |
|---|---|
| Function | dance, radio, wedding, devotional, film, club, live band, social video |
| Rhythm | groove cell, subdivision, tempo-feel, percussion role |
| Harmony | loop, modal center, functional pop changes, dominant pull, drone |
| Melody | vocal ornament, call-response, chant, melisma, range, hook length |
| Language/prosody | syllable density, code-switching, rhyme placement, vowel shape |
| Ensemble | live-band idiom, electronic production, accordion/brass/strings/synths/percussion |
| Form | strophic, verse/chorus, drop, call-response, dance break, long vamp |
| Currentness | chart signal, local playlist, short-form trend, practitioner interview |

## Regional Mexican / corridos tumbados / norteño-banda-pop lane

Composition variables:

- story-forward vocal phrasing; lyric narrative often carries identity
- guitar or bajo sexto-like rhythmic identity in many lanes
- accordion/brass/banda or sierreño-type timbral associations depending on reference
- minor-key and modal color can appear, but do not assume every track is dark
- modern crossovers may borrow trap/hip-hop low-end, vocal phrasing, or release strategy

Agent caution:

- Ask whether the user means traditional corrido, modern corridos tumbados, banda-pop, sierreño, norteño, or a crossover.
- Do not invent Spanish lyrics unless the user wants draft text and language confidence is adequate.
- For current references, check regional charts, Latin charts, local press, and actual recordings.

Pair with `latin-pop-and-reggaeton.md`, `country-americana.md`, `instrument-idiom/guitar.md`, and `../research/web-music-trend-research.md`.

## Brazilian regional pop beyond the main guide

This skill already has `brazilian-pop-and-funk.md`. Use this starter map when the user names specific scenes such as sertanejo, pagode, arrocha, piseiro, forró-pop, brega-funk, tecnobrega, or axé-adjacent pop.

Composition variables:

- Portuguese prosody is central; do not map Spanish-language Latin pop assumptions onto Brazilian music.
- Many scenes are dance/social-function driven, so groove and vocal participation matter as much as harmony.
- Live-band instrumentation and electronic production can coexist.
- Repetition, call-response, and crowd-singable hooks may be more important than harmonic novelty.

Agent caution:

- Ask for a reference lane because Brazilian scenes vary sharply by region and function.
- Separate funk carioca from samba/pagode/sertanejo/forró-related idioms.
- Use local Portuguese searches and current playlists for fresh references.

Pair with `brazilian-pop-and-funk.md`, `../fundamentals/prosody-and-language.md`, and `../research/regional-trend-evolution-analysis.md`.

## Raï / Maghreb pop / Arabic club-pop subscenes

Composition variables:

- vocal ornament, phrase shape, and language delivery are central
- electronic club grids can sit under region-specific melodic behavior
- maqam-adjacent colors may appear, but the scene should not be reduced to a scale
- call-response, percussion, and synth timbre often define the modern lane

Agent caution:

- Ask whether the user wants raï, Algerian/Moroccan/Tunisian pop, Arabic club-pop, or diaspora crossover.
- Do not claim maqam accuracy without practitioner-level verification.
- Treat references as necessary; this area is too broad for generic advice.

Pair with `mena-pop.md`, `../techniques/microtonal.md`, and `../research/reference-track-digging.md`.

## Turkish arabesk-pop / Anatolian-pop-adjacent modern styles

Composition variables:

- melodic contour and ornament often carry emotional identity
- minor/modal color, chromatic pull, and strong vocal delivery can be more important than Western chord novelty
- modern productions may combine electronic drums, strings, bağlama-like colors, and pop form
- long emotional build or intense refrain can be central

Agent caution:

- Ask whether the user means classic arabesk influence, modern Turkish pop, rock/Anatolian influence, or club crossover.
- Do not flatten Turkish makam-informed practice into equal-tempered minor scale vocabulary.

Pair with `mena-pop.md`, `folk-roots-and-traditions.md` as a legacy route, `../techniques/microtonal.md`, and current research files.

## Punjabi pop / bhangra-pop / South Asian crossover

Composition variables:

- dhol/dholak-like rhythmic identity and strong dance function in many bhangra-derived lanes
- vocal hooks may rely on short, high-energy call phrases
- Punjabi pop can cross with hip-hop, EDM, filmi, and diaspora club production
- lyric language, pronunciation, and social function matter heavily

Agent caution:

- Separate Punjabi pop/bhangra-pop from Hindi film-pop and from classical raga-derived writing.
- Ask for a reference if the user wants modern diaspora production.

Pair with `south-asian-film-pop.md`, `hip-hop-rnb.md`, `electronic-edm.md`, and `../fundamentals/prosody-and-language.md`.

## Local African dance scenes beyond Afrobeats/amapiano

Examples: gqom, singeli, gengetone, kuduro, coupé-décalé, alte-adjacent scenes, local house/club hybrids.

Composition variables:

- groove, tempo-feel, and dance function are usually primary
- loop architecture may matter more than chord progression
- percussion sound choice and rhythmic density can define the style
- vocal/chant placement often responds to dance and social function

Agent caution:

- Do not route everything through "Afrobeats" or amapiano.
- Use local/regional references and current scene names.
- Ask whether the user wants a respectful influence, a club-functional track, or a pop crossover.

Pair with `afrobeats-and-amapiano.md`, `electronic-edm.md`, `rhythm-groove/groove-and-feel.md`, and `../research/web-music-trend-research.md`.
