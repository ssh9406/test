# Folk and World Music

> **Snapshot note:** Last content review snapshot: 2026-04-27. This legacy redirect is not an active genre guide; verify current or tradition-specific claims through the active linked files.

> **Compatibility redirect:** This legacy broad-label file is kept so older links do not break. For active guidance, use `folk-roots-and-traditions.md`.

The older label "world music" is too broad for precise composition advice. When the user asks for this, route to a more specific tradition, region, scene, function, or reference track.

Use:

- `folk-roots-and-traditions.md` for folk/roots/tradition-specific writing
- `regional-scene-starters.md` for smaller regional popular scenes
- `korean-traditional.md` for gugak / K-trad fusion
- `mena-pop.md` for MENA pop / maqam-adjacent fusion
- `south-asian-film-pop.md` for South Asian film-pop / raga-derived pop
- `afrobeats-and-amapiano.md` for Afrobeats/Afropop/amapiano lanes
- `brazilian-pop-and-funk.md` for Brazilian pop/funk/MPB-adjacent writing
- `minor-and-hybrid-genres.md` for unstable or internet-native genre labels

Do not treat a tradition as a generic color. Name the source and extract concrete musical variables: rhythm, ornament, language/prosody, instrument idiom, form, function, and collaboration needs.
