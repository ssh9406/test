# Bass

Use this file when the user asks for basslines, electric bass, upright bass, synth bass, walking bass, root movement, low-end arrangement, or bass/drum lock.

## Core idea

Bass defines harmonic ground and physical pulse. A good bassline is not just the chord root; it tells the listener where the body and harmony are going.

```text
Bass part = root logic + rhythm + register + relationship to kick
```

## Range and register

Standard 4-string electric bass:

```text
E1 A1 D2 G2
```

5-string bass often adds low B0. Upright bass is commonly written sounding an octave lower than notated.

| Register | Use | Caution |
|---|---|---|
| B0-E1 | sub weight, modern low-end impact | pitch can become indistinct |
| E1-C2 | main bass foundation | conflicts with kick if too busy |
| C2-G2 | melodic fills, walking clarity | may thin out the low end if overused |
| G2+ | solos, fills, counterline | can fight vocal/guitar/piano midrange |

## Bassline roles

| Role | Description |
|---|---|
| Root anchor | supports harmony clearly; common in pop/rock/country |
| Root-fifth motion | stable folk/country/rock motion |
| Walking bass | stepwise/chromatic connection, jazz/blues/swing |
| Riff bass | memorable repeated figure, funk/rock/metal/synth-pop |
| Counter-melody | independent line against vocal/harmony |
| Sub bass | felt more than heard; EDM/hip-hop/pop |
| Pedal point | one bass note under changing chords |

## Writing by genre

| Style | Bass priority |
|---|---|
| Pop | simple hook-supporting motion, tight with kick |
| Rock | root/5th/octave drive, lock with drums/guitars |
| Punk | relentless 8ths, chord-root clarity, occasional pickups |
| Metal | double guitar riff, kick accents, low pedal precision |
| Country | root/fifth, walkups into chord changes, supportive tone |
| Jazz | walking quarter notes, guide-tone direction, swing feel |
| Funk/R&B | syncopated riff, ghost-note feel, space as groove |
| Gospel | walkups, passing tones, shout/vamp energy |
| EDM/hip-hop | sub rhythm, kick relationship, repetition and sound envelope |

## Connection devices

### Diatonic walk

```text
C → D → E → F under C → F
Effect: smooth lift into IV.
```

### Chromatic approach

```text
C → C# → D
Effect: pushes into Dm or D.
```

### Fifth motion

```text
C → G → C or C → F
Effect: stable, country/folk/rock-friendly.
```

### Enclosure

```text
Approach target D from E and C# before landing.
Effect: jazz/gospel pull.
```

### Pedal

```text
Bass holds C while chords change above: C - F/C - G/C - C
Effect: grounded, cinematic, worship/pop build.
```

## Kick relationship

Common choices:

- **Unison with kick**: tight, modern, punchy.
- **Bass fills kick gaps**: groovier, funk/R&B feel.
- **Bass long notes under busy kick**: stable but can feel static.
- **Kick marks downbeats, bass syncopates**: danceable push.

If the track feels muddy, simplify bass rhythm before changing the chords.

## Quick examples

### Country walkup

```text
Key: G, progression G → C
Bass: G - A - B - C
Use: line into IV.
```

### Pop chorus bass

```text
Progression: C - G - Am - F
Bass: C C G G | G G D G | A A E A | F F C F
Effect: clear roots with mild motion.
```

### Funk/R&B seed

```text
Center: E minor
E - rest - E G - A - G - E
Leave space; let drums answer.
```

### Metal pedal

```text
E pedal eighths, accents with guitar chugs.
Only leave pedal for section changes or fills.
```

## Common pitfalls

- **Do not make the bass constantly melodic under a busy vocal.** Support first.
- **Do not ignore the kick drum.** Bass rhythm without kick context is incomplete.
- **Do not put every chord tone in the bassline.** Root direction and groove matter more.
- **Do not overuse the lowest notes.** Very low notes lose pitch clarity if not arranged carefully.
- **Do not write upright bass like electric bass without considering sustain and attack.**
