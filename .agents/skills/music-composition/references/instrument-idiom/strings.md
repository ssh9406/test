# Strings

Use this file when the user asks for violin, viola, cello, double bass, string quartet, string section, bowed texture, pizzicato, divisi, or string playability.

## Core idea

Strings are built for sustain, phrasing, resonance, and register color. They can also be percussive, but their strongest idiom is shaped motion over time.

```text
String writing = register + bowing + fingering/open strings + texture + divisi plan
```

## Family reference

| Instrument | Lowest open string | Common role |
|---|---|---|
| Violin | G3 | high melody, agile line, shimmer |
| Viola | C3 | inner warmth, dark melody, texture glue |
| Cello | C2 | lyrical tenor/bass, rich melody, foundation |
| Double bass | E1 sounding | low foundation, weight, pizz/arco bass |

## Register color

| Register | Effect |
|---|---|
| Low strings | weight, darkness, intimacy, grit |
| Middle register | warmth and blend |
| High register | intensity, shine, tension, fragility |
| Open strings | resonance and folk/classical clarity |
| Sul ponticello / near bridge | glassy, tense color |
| Sul tasto / over fingerboard | soft, veiled color |

## Bowing and articulation

| Technique | Composition use |
|---|---|
| Legato | long lyrical lines, pads |
| Detaché | clear separated notes without harshness |
| Staccato / spiccato | light rhythmic drive |
| Tremolo | suspense, shimmer, crescendo support |
| Pizzicato | short plucked rhythm; not true sustain |
| Marcato | strong accents, cinematic rhythm |
| Glissando | expressive slide, transition, effect |
| Harmonics | fragile high color |
| Double stop | two-note sonority; check playability |

## Section writing

### Pad / sustain

- spread voices across the section
- avoid low close voicings
- move inner voices slowly
- use dynamics and bow changes to create life

### Rhythmic ostinato

- keep bowing comfortable
- use repeated patterns with slight contour
- leave room for accents and phrase breaths
- consider spiccato or tremolo depending on tempo

### Lyrical melody

- violin/cello are natural melodic carriers
- viola can be beautiful for darker, intimate melodies
- mark phrase shape more than every tiny nuance
- allow bow direction changes at musical phrase points

### Divisi

Divisi splits section players among multiple notes. It makes richer chords possible but reduces strength per note.

```text
Violins divisi into 3 notes = softer per note than all violins on one line.
```

Use divisi for lush pads; avoid it when you need maximum unison power.

## Double-stop caution

Double stops are idiomatic only when the interval and fingering work on adjacent strings. Good starting choices:

- open-string drones
- 3rds/6ths in moderate tempo
- perfect 5ths across adjacent strings
- simple two-note folk/country/fiddle shapes

Avoid fast chromatic multi-stop writing unless you know the instrument well.

## Pizzicato writing

- Pizzicato decays quickly; it does not sustain like harp or guitar.
- Fast pizzicato can be awkward depending on tempo and string crossing.
- Low pizzicato bass/cello is punchy but needs space.
- Bartók pizzicato is a special snap effect; use sparingly.

## String textures

| Texture | Use |
|---|---|
| Unison line | power, clarity, cinematic sweep |
| Octaves | broad romantic/film sound |
| Divisi pad | lush harmony |
| Tremolo chord | suspense/build |
| Pizzicato ostinato | light pulse, comedy, chamber pop |
| Counterline | emotional answer to vocal |
| Drone | folk, modal, ancient/ritual association when used specifically |

## Quick examples

### Warm pop string pad

```text
Cello: root + fifth, long notes
Viola: 3rd or 7th in middle register
Violins: 9th/top melody note, soft legato
Move one voice at a time.
```

### Cinematic lift

```text
Low strings: ostinato in 8ths
Violins: long rising line
Violas/celli: inner counter-motion
Add tremolo only near the climax.
```

### Country/folk fiddle response

```text
Short 2-bar answer after vocal line.
Use open strings, slides, and double-stop hints.
Do not overfill the vocal.
```

## Common pitfalls

- **Do not write every string pad as a keyboard chord.** Voice-leading and bowing matter.
- **Do not overuse divisi if you need power.** Divisi thins each note.
- **Do not expect pizzicato to sustain.** It decays.
- **Do not write impossible double stops casually.** Check strings/fingering.
- **Do not keep strings in one dynamic forever.** Bowed sustain needs shaping.
