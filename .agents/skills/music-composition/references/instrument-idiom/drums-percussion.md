# Drums and Percussion

Use this file when the user asks for drum grooves, fills, kit writing, percussion layers, groove energy, playable drum parts, or rhythm-section arrangement.

## Core idea

Drums organize time, energy, and section identity. A drum part is not just a beat; it tells the listener how to move and when the form changes.

```text
Drum part = pulse + backbeat/anchor + subdivision + fills + density arc
```

## Drum kit roles

| Element | Typical role |
|---|---|
| Kick | low pulse, dance/body accent, bass relationship |
| Snare | backbeat, impact, phrase punctuation |
| Hi-hat | subdivision, groove texture, openness/closure |
| Ride | sustained time feel, jazz/rock chorus lift |
| Toms | fills, tribal/arena energy, section contrast |
| Crash | section marker, chorus entry, climax |
| Ghost notes | groove detail, funk/R&B/soul feel |

## Groove building blocks

### Backbeat

```text
Snare on 2 and 4.
Use: pop, rock, country, R&B, many dance styles.
```

### Four-on-floor

```text
Kick on every quarter note.
Use: dance-pop, EDM, disco, worship/arena builds.
```

### Half-time

```text
Snare on 3 instead of 2 and 4.
Use: heavy chorus, breakdown, trap/R&B weight.
```

### Double-time feel

```text
Perceived subdivision speeds up while tempo stays same.
Use: punk/hardcore lift, chorus acceleration.
```

### Shuffle / swing

```text
Triplet-based subdivision.
Use: blues, country, roots gospel, jazz, some rock.
```

### Train beat

```text
Fast snare/brush pattern suggesting locomotive motion.
Use: country, folk-rock, Americana.
```

## Fill design

A fill should answer a formal question:

| Need | Fill type |
|---|---|
| mark section change | short tom/snare fill into downbeat |
| create suspense | rest before downbeat or rising fill |
| intensify chorus | busier fill + crash entry |
| cue breakdown | stop-time hit, silence, or single snare flam |
| avoid clutter | tiny pickup fill or no fill |

Good fills often start simple and become more active only where the form needs it.

## Density arc

A common arrangement arc:

```text
Verse: kick + hat, light snare
Pre: add ghost notes or toms
Chorus: full snare/crash/ride or open hats
Bridge: strip down or change surface
Final chorus: full kit + extra percussion
```

Do not use full cymbal wash everywhere. Cymbals can hide vocals, guitars, and consonants.

## Percussion layers

| Layer | Use |
|---|---|
| Shaker | continuous subdivision, brightness |
| Tambourine | chorus lift, gospel/pop brightness |
| Claps | crowd/community feel, pop backbeat |
| Conga/bongo | hand-percussion groove, Latin/pop contexts |
| Pandeiro/tamborim/surdo-like roles | Brazilian-informed contexts; use with specificity |
| Timpani / orchestral bass drum | cinematic impact |
| Snare roll / cymbal swell | transition build |
| Taiko-like ensemble drums | epic media scoring; avoid vague cultural shorthand |

Name the rhythmic function if you are not sure of the specific tradition. For serious tradition-based percussion, consult practitioners and current references.

## Human playability

A drummer has four limbs, but coordination is style-specific. MIDI can create patterns that no drummer would choose.

Check:

- Are kick, snare, hat/ride, and toms playable by two hands/two feet?
- Does the drummer have time to move from hat to tom to crash?
- Are ghost notes realistic at the tempo?
- Does the part require constant maximum velocity?
- Are fills placed where a player would naturally breathe with the band?

## Groove by style

| Style | Starting drum logic |
|---|---|
| Pop | clear kick/snare pattern, controlled hats, section lifts |
| Rock | strong backbeat, fills into sections, crash support |
| Punk | fast hats/ride, direct snare, few ornamental details |
| Metal | kick precision, cymbal control, riff accents, breakdown contrast |
| Country | train beat, shuffle, simple backbeat, supportive fills |
| Gospel | responsive fills, shout energy, 6/8 or shuffle options |
| R&B/funk | ghost notes, syncopated kick, pocket over density |
| Brazilian pop/funk | groove-specific percussion loop; verify substyle |
| Film/game | dynamic cueing, orchestral percussion, hybrid hits |

## Quick examples

### Pop verse to chorus

```text
Verse: kick on 1/3, snare on 2/4, closed hats 8ths
Pre: add 16th hat lift
Chorus: open hats or ride, crash on downbeat, stronger kick pattern
```

### Metal breakdown

```text
Half-time snare on 3.
Kick follows guitar chugs.
Use rests before major hits.
```

### Country train beat

```text
Snare/brush subdivision suggests forward motion.
Kick supports 1 and 3 or root/fifth bass motion.
```

## Common pitfalls

- **Do not write drums as constant fills.** Groove first, fills second.
- **Do not make every chorus louder by adding cymbals only.** Change rhythm, register, or arrangement too.
- **Do not ignore the bassline.** Kick and bass are a pair.
- **Do not use culturally specific percussion as wallpaper.** Know the function or keep it generic.
- **Do not overhumanize by randomizing everything.** Groove has intentional consistency.
