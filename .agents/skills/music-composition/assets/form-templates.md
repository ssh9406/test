# Form Templates

Quick-reference templates for common forms in classical and popular music. Use these as starting structures, not commandments. For full discussion of these forms, see `form/classical-forms.md` and `form/popular-song-forms.md`. For energy arcs across these structures, see `form/narrative-and-transitions.md`.

## Classical forms

### Binary (AB)

```
||: A :||: B :||
   I → V       V → I
```

Each section repeated. Total: ~16-32 bars. Used in Baroque dances, folk songs.

### Rounded binary

```
||: A :||: B - A' :||
   I → V    V → I → I
```

The "A return" at the end of B. Total: ~24-32 bars. Common in classical period.

### Ternary (ABA)

```
A         B          A
I → I    contrast    I → I
```

Three balanced sections. The middle section (B) is often in a related key (relative or parallel minor/major, V, etc.). Used in slow movements, character pieces, da capo arias.

### Sonata form

```
EXPOSITION (often repeated)
  Theme 1 in tonic (I)
  Transition (modulating)
  Theme 2 in V (or III in minor mode)
  Closing material in V (or III)

DEVELOPMENT
  Modulating; fragments of themes; sequences; tension building
  Retransition (long V chord preparation)

RECAPITULATION
  Theme 1 in tonic
  Transition (rewritten to stay in tonic)
  Theme 2 in tonic (the structural payoff)
  Closing material in tonic

CODA (optional)
  Final affirmation; sometimes "second development"
```

Used in: first movements of classical sonatas, symphonies, string quartets, concertos. Also common in some last movements.

### Five-part rondo (ABACA)

```
A (refrain in I)
B (episode, often in V or relative)
A (refrain in I)
C (episode in distant key — IV, parallel mode, etc.)
A (refrain in I)
```

Lively; often used as finale.

### Seven-part rondo (ABACABA)

```
A - B - A - C - A - B - A
I   V   I  IV  I   V   I
```

A more symmetric rondo. The first "A-B-A" mirrors the last.

### Sonata-rondo

```
A B A C A B A
exposition   develop   recapitulation
```

Hybrid: A and B function as exposition; C as development; final A-B-A as recapitulation.

### Theme and variations

```
Theme (T) - Var I - Var II - Var III - ... - Final Variation - (Coda)
```

Each variation is the same length and harmonic structure as the theme. Variations get progressively more elaborate; often a slow variation late; final variation virtuosic.

### Through-composed

```
A - B - C - D - E - ...
```

No section repeats. Each is new material. Used in dramatic art songs, opera scenes, programmatic works.

### Strophic (multiple verses, no chorus)

```
Intro - V1 - V2 - V3 - V4 - Outro
```

Each verse uses the same melody with different lyrics. Common in folk songs, hymns.

### Sonata cycle (3-movement)

```
Movement 1: Sonata form, fast, in tonic
Movement 2: Slow, in related key (subdominant or relative)
Movement 3: Rondo or sonata-rondo, fast, in tonic
```

Common for Classical-era sonatas and concertos.

### Sonata cycle (4-movement)

```
Movement 1: Sonata form, fast, in tonic (sometimes with slow intro)
Movement 2: Slow, in related key
Movement 3: Minuet (or Scherzo) and Trio, in tonic, moderate-fast
Movement 4: Sonata, rondo, or sonata-rondo, fast, in tonic
```

Common for Classical-era symphonies and string quartets. Order can vary (Beethoven 9 has scherzo 2nd, slow 3rd).

### Suite

```
Movement 1: Allemande
Movement 2: Courante
Movement 3: Sarabande
Movement 4: (optional) Galanteries (minuet, gavotte, bourrée)
Movement 5: Gigue
```

Baroque dance suite. All movements typically in the same key.

### Concerto sonata form (1st movement)

```
EXPOSITION 1 (orchestra alone, in tonic)
  Theme 1, Theme 2 — both in tonic

EXPOSITION 2 (with soloist)
  Theme 1 in tonic (with solo)
  Transition (modulating)
  Theme 2 in V (with solo)
  Closing in V

DEVELOPMENT (soloist + orchestra)

RECAPITULATION
  Themes in tonic
  Cadenza (extended solo improvisation, near the end)
  Final orchestral close
```

The "double exposition" is the concerto's distinguishing feature.

### Fugue

```
EXPOSITION
  Voice 1: Subject (in I)
  Voice 2: Answer (in V) + Voice 1: Countersubject
  Voice 3: Subject (in I) + Voice 2 + Voice 1
  (etc., for as many voices)

EPISODE
  Modulating; based on motifs from subject and countersubject

MIDDLE ENTRIES
  Subject in related keys (vi, iv, etc.)

EPISODE / DEVELOPMENT
  More episodes; possible stretto

FINAL ENTRY
  Subject in tonic, often with augmentation or pedal point

CADENCE
```

## Popular song forms

### Verse-Chorus (modern pop default)

```
Intro (4-8 bars)
Verse 1 (8-16 bars)
Pre-Chorus (4-8 bars)
Chorus (8-16 bars)
Verse 2 (8-16 bars)
Pre-Chorus (4-8 bars)
Chorus (8-16 bars)
Bridge (8 bars)
Final Chorus (16-32 bars; possibly modulated)
Outro (4-8 bars)
```

Total: 3:00-3:30 at 120 BPM.

### Verse-Chorus with post-chorus (modern pop)

```
Intro
Verse 1
Pre-Chorus
Chorus
Post-Chorus
Verse 2
Pre-Chorus
Chorus
Post-Chorus
Bridge
Final Chorus
Post-Chorus (extended)
Outro
```

The post-chorus often features the most memorable hook.

### AABA (32-bar form / Tin Pan Alley)

```
A (8 bars): main theme
A (8 bars): repeat with slight variation or new lyrics
B (8 bars): bridge — contrasting melody, often modulates
A (8 bars): return
```

Used in jazz standards, Tin Pan Alley, classic Broadway.

### 12-bar blues

```
| I  | I  | I  | I  |  (or I-IV-I-I in "quick change")
| IV | IV | I  | I  |
| V  | IV | I  | I  |  (or V or turnaround in last bar)
```

The simplest blues form. Used as both song structure and improvisation framework.

### Jazz blues (standard reharmonization)

```
| I7  | IV7   | I7    | I7 (V7/IV) |
| IV7 | ♯iv°7 | I7    | VI7        |
| ii7 | V7    | iii vi| ii V        |
```

The 12-bar blues with secondary dominants and ii-V's added.

### 16-bar blues

```
| I  | I  | I  | I  |
| I  | I  | I  | I  |  (twice the I section)
| IV | IV | I  | I  |
| V  | IV | I  | I  |
```

A 4-bar extension of the 12-bar pattern.

### Strophic

```
Verse 1 - Verse 2 - Verse 3 - ...
```

All verses use the same melody. Each verse has different lyrics. Common in folk, hymns, narrative ballads.

### EDM standard form

```
Intro (16-32 bars)
Build 1 (16 bars)
Drop 1 (16-32 bars)
Breakdown (16-32 bars)
Build 2 (16 bars)
Drop 2 (16-32 bars; same as Drop 1 or with variation)
Outro (16 bars; fading down)
```

Total: 5-7 minutes. Designed for DJ mixing.

### EDM radio edit

```
Intro (4-8 bars)
Verse / Build (16 bars)
Drop (16 bars)
Breakdown (16 bars)
Build (8 bars)
Drop (16 bars)
Outro (4-8 bars)
```

Shortened to 3-4 minutes for radio.

### Trap / Hip-Hop modern

```
Intro (4-8 bars)
Verse 1 (16 bars)
Hook / Chorus (8 bars)
Verse 2 (16 bars)
Hook / Chorus (8 bars)
Verse 3 (16 bars; sometimes featured artist)
Hook / Chorus (8 bars)
Outro (variable)
```

The 16-bar verse is the standard ("dropping a 16").

### K-pop standard

```
Intro (4-8 bars)
Verse 1 (8 bars; often rapped)
Pre-Chorus (8 bars; melodic)
Chorus (8-16 bars)
Post-Chorus / Drop (8 bars)
Verse 2 (8 bars; rap)
Pre-Chorus (8 bars)
Chorus (8-16 bars)
Post-Chorus (8 bars)
Bridge (8-16 bars)
Final Chorus (16-24 bars; often modulated up)
Outro (4-8 bars)
```

Total: 3:00-3:30. Highly compressed.

### Country verse-chorus

```
Intro (instrumental)
Verse 1 (16 bars)
Chorus (8-16 bars; title hook at end of phrases)
Verse 2 (16 bars)
Chorus (8-16 bars)
Bridge (8 bars; optional)
Chorus (8-16 bars; possibly with variation)
Outro (instrumental, often fading)
```

### R&B / soul ballad

```
Intro
Verse 1 (often with extended melismatic vocal)
Pre-Chorus
Chorus (with multiple ad-libs and runs)
Verse 2
Pre-Chorus
Chorus
Bridge (often modulated or with key shift)
Final Chorus (extended; vocal flourishes; outro tag)
```

## Trailer music

```
Cold open (0:00-0:15): atmospheric pad
Slow build (0:15-0:30): adding strings, sub bass, accents
Medium build (0:30-0:45): full orchestra entering
First peak (0:45-1:00): drop with full orchestra
Brief breakdown (1:00-1:15): suspended chord or atmospheric
Final peak (1:15-1:30): massive — full orchestra + choir + max percussion
Cool-down (1:30-end, optional)
```

Total: 60-120 seconds.

## Film cue (basic)

```
Statement (0-30s): theme introduced
Development (30s-1:00): theme transformed, intensifying
Climax (1:00-1:30): full orchestra, theme peaked
Resolution (1:30-2:00): release, possible callback to theme
```

## Multi-part / progressive

```
Part 1 (Movement 1): atmospheric intro
Part 2 (Movement 2): driving passage
Part 3 (Movement 3): contemplative slow section
Part 4 (Movement 4): build to climax
Part 5 (Movement 5): release / return
```

Used in prog rock, progressive metal, longer EDM, post-rock.

## Quick decision matrix

| Want | Use |
|------|-----|
| Modern pop hit | Verse-pre-chorus-chorus(-postchorus) format |
| Classical homage | Sonata form, ternary, or theme-and-variations |
| Standalone instrumental | Through-composed or ABA |
| Jazz standard | AABA in 32 bars, II-V-I-driven |
| Blues | 12-bar blues |
| Singer-songwriter / folk | Strophic or simple verse-chorus |
| Hip-hop track | 16-bar verses with 8-bar choruses |
| EDM club track | 5-7 min with build-drop-breakdown structure |
| EDM radio edit | 3-4 min compressed version |
| K-pop / J-pop | Multi-section: verse-prechorus-chorus-postchorus |
| Trailer music | 60-120 second arc with build-peak-build-bigger peak |
| Film cue | Match form to scene length and dramatic shape |
| Concept album track | Through-composed or extended form with multiple sections |
