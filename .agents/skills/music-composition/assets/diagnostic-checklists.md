# Diagnostic Checklists

Use this file when the user gives a fuzzy creative problem: "boring", "weak", "generic", "awkward", "not emotional", "too cluttered", "doesn't hit". Diagnose before suggesting fixes.

## Fast diagnostic workflow

1. Restate the problem as a musical hypothesis.
2. Check the highest-impact categories first: melody, harmony, rhythm, form, arrangement, lyric/prosody.
3. Suggest **one small test** before a full rewrite.
4. Offer 2–3 fixes with different trade-offs.

```md
Likely diagnosis: [one sentence]
Fastest test: [change one variable]
Fix options: [2–3 concrete choices]
Risk: [what could get worse]
```

## Chorus does not hit

| Check | Symptom | Fix |
|---|---|---|
| Melodic range | Chorus peak is same or lower than verse | Raise hook peak by a 3rd/4th/5th, or hold a longer note |
| Hook timing | Main hook arrives too late | Put a title/hook fragment in the first 1–2 bars of chorus |
| Pre-chorus tension | Chorus arrives without setup | Use V, V/vi, rising sequence, pedal V, or drum fill before chorus |
| Harmonic reset | Chorus loop feels like verse loop | Start chorus on I or IV if verse starts vi/i; or invert the verse loop |
| Arrangement density | No new layer/register enters | Add high octave, wider pad, cymbal, group vocal, or bass register change |
| Rhythm | Chorus is as busy as verse | Simplify melody rhythm or make drums more regular/countable |
| Lyric | Hook word is not emphasized | Put the key word on the highest, longest, or most repeated note |

Fast test: mute two verse layers, then unmute/add them at chorus. If the chorus suddenly works, the problem was contrast, not chords.

## Verse feels boring

| Check | Symptom | Fix |
|---|---|---|
| Phrase shape | Every line has same contour | Change line 3: higher range, shorter rhythm, or answer phrase |
| Harmonic rhythm | One loop repeats without lyric payoff | Change bass note, inversion, or chord on the final line only |
| Groove | Drums repeat exactly | Add pickup, ghost note, or percussion answer at bar 4/8 |
| Lyric perspective | No new information per line | Make each line reveal a new detail, image, or contradiction |
| Register | Vocal stays in same 3–4 notes | Use a lower first half and a higher answer phrase |

Fast test: keep the chords, rewrite only the final two bars of the verse.

## Melody is forgettable

| Check | Symptom | Fix |
|---|---|---|
| Motif | No repeated cell | Create a 2–5 note cell and repeat it with variation |
| Contour | Mostly flat or random | Use one clear peak and one clear descent |
| Rhythm | All notes same length | Add one longer anchor note or a syncopated pickup |
| Interval identity | Only stepwise motion | Add one memorable leap, then resolve by step |
| Hook word | Important word falls on weak note | Move it to a strong beat, long note, or high note |

Fast test: sing the melody with only rhythm on one pitch. If the rhythm alone is unmemorable, fix rhythm first.

## Harmony feels generic

| Check | Symptom | Fix |
|---|---|---|
| Loop default | I-V-vi-IV or vi-IV-I-V feels too exposed | Change bass inversions, harmonic rhythm, or final chord |
| No color tones | Triads only, same register | Add add9, sus2/sus4, 6, maj7, m9 where genre-appropriate |
| No section contrast | Verse/pre/chorus use same harmonic identity | Use modal mixture in pre or a brighter tonic start in chorus |
| No cadence | Loop never points anywhere | Add V, V/vi, ii-V, or backdoor cadence before arrival |
| Too many tricks | Every chord is chromatic | Simplify the loop and save one color chord for the lyric payoff |

Fast test: keep the same Roman numerals but change voicing and bass. If it improves, do not reharmonize yet.

## Transition is awkward

| Check | Symptom | Fix |
|---|---|---|
| Too abrupt | New section feels pasted in | Add pickup, fill, riser, common tone, or dominant preparation |
| Too prepared | Transition loses surprise | Cut the last bar, hard-stop, or use direct modulation |
| Key relation unclear | Modulation feels random | Use pivot chord, common tone, V/new key, or sequence |
| Density jump | Too many layers enter at once | Stage entrances across 2–4 bars |
| Groove mismatch | Drums switch without handoff | Add shared percussion pattern or transition fill |

Fast test: loop the last two bars before the transition and first two bars after it. Fix only the join.

## Arrangement sounds muddy or cluttered

| Check | Symptom | Fix |
|---|---|---|
| Register collision | Piano/guitar/synth/vocal all in same octave | Move one part up/down or thin voicing |
| Too many sustained parts | Harmonic cloud masks melody | Make one part rhythmic, one sustained, one melodic |
| Low-end conflict | Bass, kick, left-hand piano all active | Simplify low rhythm; remove low chord tones |
| Countermelody | Competes with lead vocal | Put countermelody in gaps or higher register |
| No negative space | Every bar is full | Drop layers before important lyric/hook entries |

Fast test: remove every part that does not change the harmony, groove, or hook. Add back only what creates contrast.

## Groove does not lock

| Check | Symptom | Fix |
|---|---|---|
| Kick/bass mismatch | Bass accents fight kick | Align key bass notes with kick, then add offbeat movement |
| Backbeat weak | Snare/clap lacks anchor | Reinforce beats 2 and 4 or a half-time beat 3 |
| Hi-hat too static | Groove feels mechanical | Add velocity accents, rests, or 16th pickup |
| Swing mismatch | Parts imply different subdivisions | Decide straight, swung, triplet, or hybrid and align core parts |
| Vocal flow conflict | Vocal ignores groove accents | Move phrase starts around the groove, not always beat 1 |

Fast test: mute harmony and listen only to drums, bass, and vocal rhythm.

## Lyric or prosody sounds awkward

| Check | Symptom | Fix |
|---|---|---|
| Stress mismatch | Natural word stress falls on weak beat | Move stressed syllable to beat/accent |
| Too many syllables | Line feels rushed | Delete filler words or split line across phrases |
| Vowel issue | High note on closed/dark vowel is strained | Put open vowel on high/long note |
| Tonal language issue | Melody fights lexical tone | Simplify contour and consult language-specific guidance |
| Rhyme pressure | Line sounds forced | Loosen rhyme scheme; use slant rhyme or internal rhyme |

Fast test: speak the lyric in rhythm before singing it. If speech feels wrong, melody will not fix it.

## Bridge does not contrast

| Check | Symptom | Fix |
|---|---|---|
| Same register | Bridge sits where verse/chorus sit | Move lower for intimacy or higher for release |
| Same harmony | Bridge is another loop | Use relative key, borrowed chords, or pedal point |
| Same rhythm | No change in phrase energy | Halve or double rhythmic density |
| No narrative turn | Lyric says same thing | Change perspective, time, consequence, or emotional claim |
| Return weak | Final chorus not set up | End bridge on V, V/vi, ♭VII, or a silence/dropout |

Fast test: make the bridge either *thinner and more intimate* or *more harmonically unstable*. Middle-ground bridges often fail.

## Too close to a reference

| Check | Symptom | Fix |
|---|---|---|
| Melody contour | Same peaks and rhythm as reference | Change interval skeleton and phrase length |
| Hook phrase | Similar wording/title cadence | Rewrite lyric image and stress pattern |
| Signature riff | Recognizable instrumental cell | Replace with different rhythm or register/function |
| Arrangement cue | Same drop/fill/stab in same spot | Move cue, change instrumentation, or blend with second reference |
| Single-source imitation | Only one anchor track | Add two more references and extract shared variables instead |

Fast test: remove the reference audio. If you can still identify the source from one element, transform that element.

## Writer's block

| Symptom | Fix |
|---|---|
| Too many options | Limit to one scale, four chords, one instrument, ten minutes |
| Can't start | Write a bad 4-bar loop on purpose; revise after hearing it |
| Over-editing | Separate drafting and critique sessions |
| No direction | Choose three adjectives, one reference boundary, one technical constraint |
| Loop fatigue | Write the next section with opposite density/register/harmonic rhythm |

Fast test: create three intentionally different 8-bar sketches instead of improving one loop.
