# Session Brief and Decision Log Templates

Use these templates to keep multi-turn composition sessions coherent. Pair with `../references/creative-workflows/user-agent-collaboration.md`.

## Minimal project card

```md
**Project card**
- Goal:
- Genre frame:
- Mood:
- References:
- Key / tempo / meter:
- Current section:
- Chosen seed:
- Keep:
- Change:
- Open decision:
```

## Compact decision log

```md
**Decisions so far**
1.
2.
3.

**Rejected for now**
- 

**Can revisit later**
- 
```

```md
**User asks:** "Make it more ______."

- Form:
- Lyric/prosody:

Smallest useful edit:
```

## Handoff summary

Use this if the conversation is long or the user asks to continue later.

```md
**Current state**
- We are writing:
- Current best version:
- Main unresolved issue:
- Next best action:

**Material generated**
- Chords:
- Motif/topline:
- Groove:
- Form:
- Lyric/title:

**User preferences learned**
- Likes:
- Dislikes:
- Avoid:
```
