## 3. Mood to musical variables

| Mood target | Harmony | Melody | Rhythm | Texture |
|---|---|---|---|---|
| tender | added 6/9, gentle plagal motion | stepwise, narrow range | relaxed, fewer attacks | warm midrange, sparse layers |
| lonely | minor/add9, pedal tones | repeated notes, falling thirds | space between phrases | dry solo instrument, distant pad |
| anxious | chromatic neighbor chords, unresolved dominant | small intervals, unstable landing | syncopation, offbeat accents | thin highs, pulsing ostinato |
| triumphant | I-IV-V, Lydian/major color | rising fourth/fifth, high peak | strong downbeats | wide voicing, octave doubles |
| sensual | minor 7/9, slow dominant pull | curved contour, late resolution | behind-the-beat feel | low-mid warmth, breath/space |
| uncanny | modal mixture, tritone, mediant shifts | familiar cell in wrong context | lopsided phrase | register gaps, brittle timbre |

## 6. Constraint dice

Choose one from each row.

| Axis | Constraint options |
|---|---|
| Harmony | only 2 chords / no V chord / one borrowed chord / pedal bass / chromatic bass |
| Melody | 5 notes only / one leap only / start every phrase offbeat / highest note only once |
| Rhythm | no downbeat hook / 3+3+2 grouping / halftime chorus / one bar of silence |
| Form | chorus first / no pre-chorus / 2-bar post-hook / bridge uses verse lyric rhythm |
| Texture | no cymbals until chorus / one instrument per octave / mono verse, wide chorus |
| Lyric | title appears once / no perfect rhymes / image before explanation |

## 7. Mutation menu

Use after the user likes a seed but wants more versions.

| Mutation | What changes | What stays |
|---|---|---|
| Transpose | key/register | contour and rhythm |
| Invert | direction | rhythm and interval size roughly |
| Sequence | starting pitch/chord | shape |
| Compress | note values | pitch idea |
| Expand | longer notes / wider gaps | identity |
| Displace | metric placement | pitches |
| Reharmonize | chords under melody | melody |
| Re-orchestrate | texture/timbre | notes/chords |
| Simplify | fewer notes/chords | emotional target |

## 8. Reference triangulation card

```md
| Reference | Borrow only this variable | Do not copy |
|---|---|---|
| A | energy / tempo / density | melody, bassline, lyric |
| B | harmony density / voicing | progression order if too recognizable |
| C | vocal phrasing / section arc | artist identity, signature ad-libs |
```

## 9. Section contrast card

```md
Verse = smaller by: register / density / rhythm / harmony / lyric camera
Pre = tension by: bass climb / dominant / lyric urgency / rhythmic acceleration
Chorus = release by: hook repetition / range peak / wider voicing / simpler rhythm
Bridge = new information by: mode shift / perspective shift / texture reset
```

## 10. "Make it more X" card

| Request | First variables to try |
|---|---|
| more modern | earlier hook, shorter intro, leaner loop, current groove reference |
| more classic | stronger cadence, live-band idiom, longer phrases |
| more expensive | fewer busy fills, wider voicings, controlled register, clearer bass |
| more raw | less polish, narrower arrangement, stronger performance gestures |
| more catchy | shorter motif, title repetition, clearer rhythm, higher landing note |
| less generic | bass variation, phrase asymmetry, one surprising chord, distinctive rhythm |
| less cluttered | reduce midrange layers, separate registers, simplify counterlines |
| more emotional | delay resolution, add appoggiatura, raise chorus peak, expose vocal |
