# Progressions Catalog

A quick catalog of common chord progressions across genres. Use this as a lookup for **playable options**, not as a replacement for listening, score study, or genre-specific idiom.

Most examples are given in C major or A/C minor. Transpose to the user's key and genre. Famous-song references are **simplified, non-authoritative recognition examples**: do not present them as exact published transcriptions unless you have checked a reliable score or chart.

For functional theory behind these progressions, see `harmony/functional-harmony.md`. For voice leading, see `harmony/voice-leading.md`. For jazz substitutions and reharmonization, see `harmony/jazz-harmony.md` and `harmony/reharmonization.md`. For modal progressions, see `harmony/modal-harmony.md`.

## How to use this catalog

1. Pick a **function**: stable loop, cadence, lift, darkening, bridge contrast, modulation setup.
2. Pick a **genre frame**: pop, rock, jazz, folk, EDM, cinematic, theatre, etc.
3. Transpose and adjust **harmonic rhythm**: one chord per bar, two per bar, pedal bass, syncopated hits.
4. Add **voicing and bass motion**. The same RN pattern can feel generic or fresh depending on inversion, register, rhythm, and texture.

| Axis | I-V-vi-IV | C-G-Am-F | Modern pop, rock choruses, singer-songwriter loops |

| 50s / doo-wop | I-vi-IV-V | C-Am-F-G | 1950s pop, ballads, nostalgic turns |

### Major key cadential

| Progression | Effect |
|-------------|--------|
| ii-V-I | Strong cadential resolution; jazz default and useful in many tonal styles |
| IV-V-I | Direct pop / rock / folk cadence |
| V-I | Authentic cadence; maximum functional closure when prepared well |
| IV-I | Plagal cadence, hymn/gospel/folk color |
| ii⁶-V-I | Smooth classical voice-leading version of ii-V-I |
| V⁷-I | Strong dominant-to-tonic resolution |
| V-vi | Deceptive cadence; avoids expected tonic arrival |

## Modal progressions

Modal progressions usually work because they **avoid over-functional dominant gravity** and emphasize the mode's characteristic degree.

### Dorian

| Progression | Example in D Dorian | Notes |
|-------------|---------------------|-------|
| i-IV | Dm-G | Signature Dorian color: minor tonic + major IV (^6) |
| i-IV-i-IV | Dm-G-Dm-G | Two-chord modal vamp |
| i-♭III-IV-i | Dm-F-G-Dm | Dorian lift with stable tonic return |
| i-v-IV-i | Dm-Am-G-Dm | Softer, more diatonic Dorian loop |

### Phrygian

| Progression | Example in E Phrygian | Notes |
|-------------|----------------------|-------|
| i-♭II | Em-F | Signature Phrygian semitone tension |
| i-♭II-i-♭II | Em-F-Em-F | Static Phrygian vamp |
| ♭II-i | F-Em | Characteristic Phrygian cadence-like resolution |
| i-♭VII-♭II-i | Em-D-F-Em | Dark modal cycle; avoid making D act like functional dominant |

### Mixolydian / rock major-minor

| Progression | Example in G | Notes |
|-------------|--------------|-------|
| I-♭VII | G-F | Signature Mixolydian/root-rock sound |
| I-♭VII-IV-I | G-F-C-G | Rock, folk-rock, guitar music |
| I-IV-♭VII-I | G-C-F-G | Plagal, open, less dominant-driven |
| I-V-IV-I | G-D-C-G | Rock major/minor blend; V major adds leading-tone pull, so it is not purely diatonic Mixolydian |

### Lydian

| Progression | Example in F Lydian | Notes |
|-------------|--------------------|-------|
| I-II | F-G | Signature Lydian color: bright major II / raised ^4 |
| I-II-I-II | F-G-F-G | Lydian vamp |
| I-V-II-I | F-C-G-F | More stable Lydian loop; keep F as center |
| I-iii-II-I | F-Am-G-F | Smooth, bright, less cadence-heavy |
| I-♯iv°-V-I | F-B°-C-F | Diatonic to F Lydian; use carefully because it can imply functional resolution |

| Progression | Example in C | Notes |
|-------------|--------------|-------|
| V/V-V-I | D-G-C | Dominant of dominant |
| V/IV-IV-I | C7-F-C | Pull toward IV; blues/folk/gospel friendly |
| V/vi-vi-IV-V-I | E-Am-F-G-C | Emotional tonicization of vi |
| V/ii-ii-V-I | A-Dm-G-C | Sets up a stronger ii-V-I |

### Modal mixture / borrowed chords

| Progression | Example in C | Notes |
|-------------|--------------|-------|
| I-iv-I | C-Fm-C | Borrowed iv from parallel minor |
| I-IV-iv-I | C-F-Fm-C | Strong bittersweet voice-leading |
| I-♭VI-♭VII-I | C-A♭-B♭-C | Parallel-minor rock/power-pop color |
| I-♭III-♭VII-IV | C-E♭-B♭-F | Rock mixture; blues/Mixolydian/minor overlap |
| vi-IV-I-V/vi | Am-F-C-E | Secondary dominant pull back to vi |

### Augmented sixth chords

| Progression | Example in C | Notes |
|-------------|--------------|-------|
| It⁶-V-I | A♭-C-F♯ → G → C | A♭ resolves down to G; F♯ resolves up to G |
| Fr⁶-V-I | A♭-C-D-F♯ → G → C | Adds D; whole-tone-like color |
| Ger⁶-I⁶/₄-V-I | A♭-C-E♭-F♯ → C/G → G → C | Often routed through cadential 6/4 to avoid parallel 5ths |

German +6 is the augmented-sixth type most often enharmonically reinterpreted as a dominant seventh. Do not describe Italian +6 as having the same pitch content as V7.

## Bass-line-driven progressions

Sometimes the harmony is best understood from the bass line rather than the chord labels.

### Descending chromatic bass / lament family

In A minor, a common bass line is:

```text
A - G♯ - G - F♯ - F - E
^1 - raised ^7 - ♭^7 - raised ^6 - ♭^6 - ^5
```

Possible chords:

```text
Am - E/G♯ - C/G - D/F♯ - F - E
```

This gives a lament-like pull: tragic, stately, inevitable. The surface chords vary widely; the bass line is the identity.

### Diatonic descending bass

```text
C - G/B - Am - G - F - C/E - Dm - G
I - V6 - vi - V - IV - I6 - ii - V
```

Useful for ballads, worship/gospel-adjacent writing, and classic pop craft.

### Ascending bass

A stepwise bass line ^1-^2-^3-^4-^5 can support many chords above it. Common in soul ballads, theatre builds, and pre-choruses.

### Pedal tonic

A held tonic in the bass while harmony moves above it. Common in classical, ambient, rock, and cinematic music. Creates stability and scale.

### Pedal dominant

A held dominant in the bass building tension toward tonic. Common in classical retransitions, EDM builds, theatre vamps, and cinematic cues.

## Specific characteristic progressions — simplified examples

Use these as **recognition anchors**, not definitive transcriptions.

### "Creep"-family chromatic mediant + borrowed iv

```text
I - III - IV - iv
C - E - F - Fm
```

The striking color is the chromatic mediant III followed by the borrowed minor iv.

### "Don't Stop Believin'"-family descending bass / Pachelbel cousin

```text
I - V6 - vi - iii6 - IV - I6 - ii - V
C - G/B - Am - Em/G - F - C/E - Dm - G
```

The identity is the stepwise bass descent and delayed return to the dominant.

### "Wonderwall"-family suspended guitar loop

```text
i - ♭III - ♭VII - IVsus
Em7 - G - Dsus4 - Asus4
```

The sus chords and common tones matter as much as the Roman numerals.

### "Hotel California"-family descending minor cycle

```text
i - V - ♭VII - IV - ♭VI - ♭III - iv - V
Bm - F♯ - A - E - G - D - Em - F♯
```

The major V at both ends reinforces the minor tonic.

### "Hey Jude"-family singalong outro

```text
I - ♭VII - IV - I
F - E♭ - B♭ - F
```

This is a Mixolydian/plagal singalong color. For the verse, avoid overclaiming a single loop; it is mostly simple F-major tonic/dominant/subdominant motion with local variations.

### Guitar-rock Mixolydian loop

```text
I - ♭VII - IV - I
D - Cadd9 - G - D
```

Open-string voicing and guitar register are part of the sound.

### "Africa"-style pop-rock modal mixture caution

Toto's "Africa" is frequently simplified differently across charts because sections, slash bass, and keyboard voicings matter. Use it as a reminder to **verify famous-song examples** before naming exact changes. For a generic similar color, try:

```text
I - iii - vi - ♭VII - IV - ii
B - D♯m - G♯m - A - E - C♯m
```

Treat that as an original pop-rock/modal-mixture palette, not a claim of an exact transcription.

## Quick decision matrix

| Want | Try |
|------|-----|
| Pop standard feel | I-V-vi-IV |
| Slightly darker pop | vi-IV-I-V |
| 1950s / nostalgic feel | I-vi-IV-V |
| Lifting / building | I-iii-vi-IV or IV-V/vi-vi-V |
| Canon-like feel | I-V-vi-iii-IV-I-IV-V |
| Modal Dorian feel | i-IV |
| Modal Phrygian feel | i-♭II |
| Modal Mixolydian / guitar-rock feel | I-♭VII-IV-I |
| Lydian brightness | I-II |
| Strong cadence | ii-V-I (jazz); IV-V-I (pop); V-I (classical) |
| Surprise | Deceptive cadence: V-vi |
| Tension / build | Pedal V or rising sequence |
| Modulation up | V/new key → new tonic |
| Descending drama | Lament bass or minor descending cycle |
| Modal mixture color | I-iv-I; I-♭VI-♭VII-I; I-♭III-♭VII-IV |
| Jazz minor cadence | iiø7-V7♭9-i |
| Backdoor soul/jazz cadence | iv-♭VII7-I |
